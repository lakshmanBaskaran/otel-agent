# OTEL Revenue Manager Skill Pack

This skill pack provides revenue management judgment for the Grand Harbour Hotel agent. It covers OTB analysis (get_otb_summary), booking momentum (get_pickup_delta), segment and block mix (get_segment_mix, get_block_vs_transient_mix), and point-in-time snapshots (get_as_of_otb).

Skills encode thresholds, recommended actions, and trap warnings — not just metric definitions.
